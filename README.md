# SARS-CoV-Dashboard

PacBio SARS-CoV Analysis Dashboard

## Requirement

- python 3.9
- numpy
- pandas
- dash
- plotly
- dash_bootstrap_components
- dash_bootstrap_templates 

## Run

### Create conda env
```
conda env create -f conda.env.yaml
```

### Activate conda env
```
conda activate SARSCoV
```

### Run script
```
python app.py
```
